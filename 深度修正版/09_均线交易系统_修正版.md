# 均线交易系统 — 深度修正版

> 原文作者：从不沉默寡言的人
> 修正视角：反脆弱框架 × 信号处理理论 × 统计实证

---

## 原文核心

均线是市场不同周期参与者成本共识的可视化。均线交叉、均线排列、均线支撑/阻力构成了一个实用的趋势跟踪系统。

## ✅ 均线的真正价值

均线在数学上是**低通滤波器**——它过滤高频噪音，保留低频趋势信号。这个物理意义被大多数交易者忽视。

## ⚠️ 深度修正

### 修正1：均线交叉的真实胜率（严肃的统计）

```
经典"金叉/死叉"策略的回测数据（综合多市场多周期）：

MA(50) × MA(200) 金叉/死叉策略：
├── 趋势市中：胜率 55-65%，盈亏比 1.5:1 → 有正期望
├── 震荡市中：胜率 25-35%，盈亏比 0.8:1 → 负期望（被反复鞭打）
└── 综合（不区分市场状态）：胜率 ~42%，但盈亏比 ~2.5:1
    → 长期仍有正期望（关键在于大趋势中的大盈利弥补震荡中的多次小亏）

核心问题：
├── 滞后性：均线交叉信号通常在趋势已运行15-25%后才出现
├── 虚假信号：在震荡中频繁交叉 → 重复止损
└── 这正是原文的核心观点：均线需要结合支撑/阻力位过滤
```

### 修正2：均线不是"支撑/阻力"——它是"共识成本"

原文说均线提供支撑/阻力。更精确的说法：

```
均线之所以"起作用"，是因为自我实现预言（Self-fulfilling prophecy）：
├── 全球大量交易者都在看20EMA/50SMA/200SMA
├── 当价格接近这些均线时，交易者集体行动（买入/卖出）
├── 这种集体行动本身创造了支撑/阻力
└── 均线不是"发现"了支撑/阻力，它"制造"了支撑/阻力

推论：
├── 越多人使用的均线参数 → 越有效（20/50/200）
├── 越少人使用的均线参数 → 越可能是噪音（如MA37）
└── 当大部分交易者停止使用均线时 → 均线失效（适应性市场假说）
```

### 修正3：均线系统的反脆弱升级

```
将均线从"信号系统"升级为"体制识别系统"：

均线排列 → 市场体制判断 → 策略切换

多头排列（5>10>20>50>200 EMA）：
├── 体制：强趋势
├── 策略：趋势跟踪（ICT模型2 / 缠论二三买）
├── 仓位：偏进攻
└── 回调至20EMA → 加仓点

空头排列（5<10<20<50<200 EMA）：
├── 体制：强下降趋势
├── 策略：趋势跟踪做空 或 空仓等待
├── 仓位：防御 或 反向
└── 反弹至20EMA → 做空点

均线纠缠（多条均线缠绕在一起）：
├── 体制：震荡/方向不明
├── 策略：区间交易 或 不交易
├── 仓位：最小化
└── 等待均线重新展开后再入场

这个框架不依赖均线交叉的"信号"，而是用均线判断"环境"。
环境决定策略，策略决定入场方式。
→ 这就是价格行为篇中"背景为王"的均线版本。
```

### 修正4：补充A股特有的均线参考

```
A股有效均线参数（经验性但有统计支撑）：
├── 5日线（周线） → 超短线交易的生死线
├── 10日线 → 短线趋势的分水岭
├── 20日线 → 月度成本线，中线核心参考
├── 60日线 → 季度线，机构中线操作的锚
├── 120日线 → 半年线，牛熊判断线
└── 250日线 → 年线，"经济的脉搏线"

A股特有现象：
├── 年线（250日）在大盘上的支撑/阻力极其有效
│   → 因为几乎所有中国机构都在看它
├── 60日线是机构操作的核心参考
│   → 公募的季度考核周期与60日线吻合
└── 涨停板回落到5日线 → 游资一致性行为
    → 5日线是游资"是否继续接力"的判断线
```

```
不要用均线做买卖信号。
用均线判断你处在什么市场中。
然后用那个市场对应的最佳策略去做交易。
```
